JavaScript Basics: A Comprehensive Guide

2024-03-28 Technology

A comprehensive guide to JavaScript fundamentals, covering variables, functions, and objects

This article provides a detailed overview of JavaScript's fundamental concepts, including variable declarations, function types, and object manipulation.

Variable Declarations

JavaScript offers three ways to declare variables:

// var - function-scoped
var name = "John";

// let - block-scoped
let age = 30;

// const - constant value
const PI = 3.14;

Functions

Functions in JavaScript can be defined in multiple ways:

// Function declaration
function sayHello(name) {
    return `Hello, ${name}!`;
}

// Arrow function
const greet = (name) => `Hello, ${name}!`;

Objects

Objects are collections of key-value pairs:

const person = {
    name: 'Alice',
    age: 25,
    sayHi() {
        console.log(`Hi, I'm ${this.name}`);
    }
};
